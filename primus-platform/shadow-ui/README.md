# shadow-ui

Experimental UI variant for research and testing.

## Overview

`shadow-ui` is an experimental implementation of the Primus UI used for:
- Testing alternative UI frameworks
- UX research and prototyping
- Component experimentation
- Feature exploration

## ⚠️ Not for Production

This module is **NOT** recommended for production use. Use `primus-ui` instead.

## Quick Start

### Build and Run

```bash
cd primus-platform/shadow-ui

npm install
npm start
```

### Access UI

```
http://localhost:4201  # Different port than primus-ui
```
